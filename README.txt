ESTAMPA STUDIO CTES — V5 PREMIUM

Cambios principales de esta versión:
- Se mantuvo el logo real y la foto real de stickers.
- Se agregó una capa visual más premium con profundidad 3D suave.
- El hero ahora tiene tarjetas flotantes: DTF Textil, Sublimación y Stickers UV.
- Se agregaron microanimaciones: luces ambientales, brillo en botones, cards con efecto 3D, hover con reflejo y barra de progreso superior al hacer scroll.
- Las tarjetas de productos, redes, stickers y remeras tienen más sensación de volumen.
- El stock real de remeras sigue cargando desde Firebase con fotos, talle/stock y precio.
- Se mantuvo el carrito, panel admin, Firebase y compra por WhatsApp.
- Las animaciones se reducen automáticamente si el dispositivo pide menos movimiento o si se usa pantalla chica.

Archivos/carpetas a subir al proyecto:
1) index.html
2) app.js
3) carpeta assets completa
4) firebase.rules, solo si todavía no actualizaste las reglas de Firestore.

Importante:
El archivo JavaScript debe llamarse exactamente app.js porque el HTML lo carga así:
<script type="module" src="app.js"></script>

En Firebase > Firestore Database > Rules, copiá el contenido de firebase.rules y publicalo.
